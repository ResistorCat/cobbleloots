Cobbleloots is a mod that adds a new way to obtain items in Minecraft. It introduces the `Loot Balls`, which can be found in different ways to obtain various items. They work similarly to the Poké Balls in the Pokémon games, where you can find special items inside them.

### Dependencies (Both Fabric and Neoforge)
Cobbleloots requires the following mods to be installed:
- [Cobblemon](https://modrinth.com/mod/cobblemon)

### Dependencies (Fabric)
- [Fabric API](https://modrinth.com/mod/fabric-api)

### Installation
<Steps>
    1. Download the latest version of Cobbleloots from [Modrinth](https://modrinth.com/mod/cobblemon-loot-balls) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/cobblemon-loot-balls).

    2. Install the required dependencies listed above.

    3. Add the downloaded JAR file to your mods folder.

    4. Launch the game and enjoy Loot Balls in your world!
</Steps>
